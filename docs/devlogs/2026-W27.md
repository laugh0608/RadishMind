## 关键决策

- `audit_store_idempotency_runtime_implementation_entry_review_defined` 只表示 idempotency runtime 实现准入已完成评审且仍 blocked，不代表 idempotency runtime ready。
- `audit_store_delivery_runtime_implementation_entry_review_defined` 只表示 delivery runtime 实现准入已完成评审且仍 blocked，不代表 delivery runtime ready。
- `audit_store_concrete_durable_backend_selection_review_defined` 只表示 backend family 静态选择为 append-only metadata audit log，不代表 storage adapter runtime、DB provider 或 audit store runtime ready。
- `audit_store_runtime_implementation_entry_refresh_v5_defined` 只表示已消费 concrete backend family selection 并复评 audit store runtime task card；结论仍 blocked，不代表 storage adapter runtime、writer / idempotency / delivery runtime、audit store runtime 或 production resolver runtime ready。
- storage adapter runtime、writer result、delivery runtime、operator approval、credential handle、backend health 和 no leakage smoke runtime 仍是后续 runtime 前置，不能用 static readiness、schema artifact、memory store、fake resolver 或 fixture 替代。
