更新时间：2026-07-20

本文档用于回答三件事：

- `RadishMind` 现在已经具备什么能力
- 还缺哪些关键能力
- 接下来应该先补哪一块，而不是继续在旧实验或假想接线上打转

如果你要先理解“项目为什么这样分层”，先读 [战略定义](radishmind-strategy.md)。
